# Jupyter Notebooks

This directory contains Jupyter notebooks demonstrating various SAS Viya Python integration scenarios.

## Notebooks

- `01_getting_started.ipynb` - Basic connection and data exploration
- `02_churn_analysis.ipynb` - Customer churn prediction walkthrough
- `03_fraud_detection.ipynb` - Real-time fraud detection implementation
- `04_time_series_forecasting.ipynb` - Advanced forecasting techniques
- `05_model_deployment.ipynb` - Model deployment and monitoring

## Usage

1. Install Jupyter: `pip install jupyter`
2. Start Jupyter: `jupyter notebook`
3. Navigate to this directory and open the desired notebook

## Prerequisites

Make sure you have:
- Configured your SAS Viya connection in `../config/sascfg_personal.py`
- Installed all required packages from `../requirements.txt`
- Access to a SAS Viya environment
